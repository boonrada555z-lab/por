# 🏥 Bangkok Hospital Phitsanulok Portal (BPH Portal)
> **ระบบพอร์ทัลรวบรวมและเข้าถึงลิงก์ใช้งานทางการแพทย์และแผนกต่างๆ อย่างอัจฉริยะ ปลอดภัย และเสถียรขั้นสูง**

## 📌 บทนำและวัตถุประสงค์ (Overview)

**BPH Portal** คือระบบ Web Application พอร์ทัลศูนย์กลางรวบรวมลิงก์การใช้งานสำคัญภายในโรงพยาบาลกรุงเทพพิษณุโลก (Bangkok Hospital Phitsanulok) เพื่อช่วยให้บุคลากรทางการแพทย์ แพทย์ พยาบาล เภสัชกร และเจ้าหน้าที่ฝ่ายสนับสนุน สามารถเข้าถึงระบบงาน ไอทีซอฟต์แวร์ และแบบฟอร์มต่าง ๆ ได้อย่างสะดวก รวดเร็ว และเป็นระเบียบเรียบร้อย ลดความซ้ำซ้อนของการจำ URL และสร้างความพร้อมในกระบวนการทำงานระดับส่วนหน้าและส่วนหลัง (OPD, IPD, IV, DIS, และฝ่ายช่วยเหลืออื่น ๆ)

---

## 🛠️ สถาปัตยกรรมระบบ (Technical Architecture)

ตัวแอปพลิเคชันประกอบไปด้วยสถาปัตยกรรมแบบ **Full-Stack (Vite + React + Express + Node.js)** ที่ออกแบบมาให้ทนทานต่ออุปสรรคทางเครือข่ายอินเทอร์เน็ต ผ่านระบบการจัดการข้อมูลแบบ **Triple-Layer Redundancy (การสำรองข้อมูลสามชั้น)**:

1. **Supabase Cloud Database (Primary Storage)**:
   - บันทึกและซิงโครไนซ์ข้อมูลหมวดหมู่ (Categories) และปุ่มลิงก์ระบบงาน (Buttons) บนคลาวด์แบบเรียลไทม์
   - ปลอดภัยสูงด้วยการจัดการสิทธิ์ผ่าน **Row Level Security (RLS)** และ **Service Role Key Verification** ข้ามพ้นการบล็อกไอพีภายนอกโรงพยาบาล
2. **Local Failproof Database Cache (Secondary Storage)**:
   - ทำงานเซิร์ฟเวอร์ด้วย `buttons-db.json` และ `categories-db.json` ภายในเครื่องทันที หากการเชื่อมต่อ Supabase ล้มเหลว หรือคีย์ API ไม่ถูกต้อง เพื่อให้บริการสามารถดำเนินต่อไปได้แบบเหนียวแน่น 100%
3. **Google Sheets TSV Bootstrap (Tertiary Layer)**:
   - มีฟีเจอร์ดึงฐานข้อมูลเมนูเริ่มต้น (Seed Data) โดยยึดความสะดวกของผู้ดูแลผ่านไฟล์ Google Sheets เพื่อประหยัดเวลาการพิมพ์ป้อนข้อมูลใหม่ทั้งหมดเมื่อเริ่มระบบ

## 🌟 ฟังก์ชันการใช้งานหลัก (Core Features)

### 1. ระบบคัดกรองลิงก์อัจฉริยะ (Dynamic Dynamic Filter)
- แยกหมวดหมู่หลักสอดคล้องกับโครงสร้างแผนกในโรงพยาบาล เช่น **OPD**, **IPD**, **IV**, **DIS**, **ผู้ช่วย** และ **อื่น ๆ**
- ค้นหาแบบเรียลไทม์ด้วยตัวกรองตัวอักษรทันใจ รองรับการค้นหาทั้งคำอธิบาย ไอคอน และคีย์เวิร์ด

### 2. ระบบตู้ทดสอบโปรโตคอลความปลอดภัย (Dynamic Routing Diagnostics Overlay)
- เมื่อคลิกใช้งานลิงก์ใด ๆ ระบบจะไม่เปิดขึ้นมาอย่างทื่อ ๆ แต่จะนำเข้าสู่ **"Cyber Diagnostics Overlay (4-Step Diagnostics Tunnel)"** ซึ่งจำลองการตรวจสอบการเชื่อมต่อและความปลอดภัยในโครงข่ายโรงพยาบาล:
  1. *Protocol Checking* (เช็กเกราะความปลอดภัย https/http)
  2. *DNS Lookup & Health Ping* (วิเคราะห์สถานะปลายทาง)
  3. *Registering Analytics Traffic* (บันทึกสถิติยอดชมยอดคลิกลงสู่ฐานข้อมูลสำมะโน)
  4. *Secure Terminal Redirection* (ส่งต่อไปทิศทางเป้าหมายอย่างปลอดภัย)
- ฟีเจอร์นี้ให้ความรู้สึกเป็นธรรมชาติ ทันสมัย น่าเชื่อถือสูง สไตล์เซฟตี้คลินิกัลไอที

### 3. ระบบจัดการหมวดหมู่และจัดการปุ่มผ่านหลังบ้านแบบเรียลไทม์ (Admin Portal Manager)
- มีระบบรักษาความปลอดภัย 3 ด่าน (**3-Step Gate Protection**): รหัสส่วนตัวของผู้ดูแลระบบ + การทวนรหัสดิจิทัล
- โหมดแอดมินสามารถเพิ่ม ลบ แก้ไข ปรับขนาดปุ่ม เปลี่ยนสี เลือกสรรไอคอนอิโมจิ ย้ายตำแหน่ง และกำหนดระดับความสำคัญ (Priority Mode) เพื่อแสดงผลที่หน้าแรกสุดได้ทันที

### 4. ระบบวิเคราะห์ข้อมูลงามตา (Rich Analytics Dashboard)
- แดชบอร์ดในตัวที่ออกแบบเฉพาะ แสดงสถิติยอดการมีส่วนร่วมและการคลิกใช้งานลิงก์แต่ละประเภทอย่างสรุปชัดเจน ถอดรูปออกมาเป็นแผนภูมิร้อยละจำลองสไตล์แดชบอร์ดล้ำสมัย

### 5. ระบบบอร์ดประกาศข้อความฉุกเฉิน (Emergency Alert Broadcast Overlay)
- ผู้ดูแลสามารถเปิดไฟสัญญาณไซเรน แสดงบอร์ดข้อความด่วนสำหรับเหตุฉุกเฉินหรือประกาศสำคัญ เช่น ประกาศระบบงานไอทีปรับปรุงประจำสัปดาห์

## 🚀 ฟังก์ชันไฮเทคสุดล้ำสมัย: AI Cyber Command & Voice Center 🎧 (NEW!)

เพื่อยกระดับความ **"ไฮเทค ทันสมัย ล้ำยุคไร้สัมผัส (Hands-free Operations)"** ระบบได้มีการเพิ่มเติมฟังก์ชัน **AI Cyber Assistant** ที่ตอบโจทย์การทำงานในโรงพยาบาลได้อย่างชาญฉลาดเป็นพิเศษ:

### 🎙️ 1. ระบบสั่งการด้วยเสียงอัจฉริยะภาษาไทย (Thai Speech Recognition Engine)
- เภสัชกร แพทย์ หรือพยาบาลที่อาจมีมือไม่ว่างในการพิมพ์ (เตรียมยา, ล้างมือไร้เชื้อ หรือใส่ถุงมือ) สามารถคลิกปุ่มไมค์หรือกดคีย์ด่วน เพื่อเปิดศูนย์ควบคุมความคุมด้วยเสียง
- **คำสั่งเสียงที่ระบบเข้าใจทันที**:
  - `ไป OPD` / `ไป IPD` / `ไป IV` / `ไปอื่นๆ` -> เพื่อสลับหมวดหมู่อย่างรวดเร็ว
  - `เปิด [ชื่อปุ่ม]` เช่น *“เปิด Google”*, *“เปิดระบบเคลม”* -> สั่งเปิดหน้าต่างระบบนั้นทันทีผ่าน Diagnostics Tunnel โดยไม่ต้องยกนิ้วสัมผัสคีย์บอร์ด
  - `ค้นหา [คำค้น]` เช่น *“ค้นหายา”* -> พิมพ์ค้นหาให้ทันทีในกล่องเซิร์ชบาร์
  - `รีเซ็ต` / `ล้าง` -> ล้างคำกรองทั้งหมด
  - `เข้าแอดมิน` -> เปิดหน้าต่างล็อกอินเพื่อตรวจสอบข้อมูลเข้าสู่ระบบทันที
- **Glowing Pulsating Audio Waveform Overlay**: แสดงคลื่นสัญญาณคลื่นเสียงนีออนสวยสะดุดตาจำลองตามระดับการพูดพร้อมเอฟเฟกต์ไฟวิ่งไซไฟ

### 🔊 2. ระบบโมดูลเสียงพูดตอบสนองอัจฉริยะ (Speech Synthesis Talkback)
- ระบบส่งเสียงพูดภาษาไทยโต้ตอบแบบดิจิทัลอย่างชัดเจนเมื่อมีการตอบรับ เพื่อบอกผลลัพธ์ เช่น *"กำลังกรองข้อมูลหมวดหมู่ OPD ให้ค่ะ..."* หรือ *"กำลังนำท่านเข้าสู่ระบบงาน..."* ให้ประสบการณ์ล้ำยุคเสมือนระบบคอมพิวเตอร์ในหนังไซไฟ (Jarvis / Siri สำหรับโรงพยาบาล)

## 🏆 ประโยชน์ของระบบต่อบุคลากรทางการแพทย์ (Key Benefits)

* **ความปลอดภัยของระบบเป็นเลิศ (High Security)**: ป้องกันอันตรายจากการลอบแก้ไขข้อมูลด้วยการล็อกระบบผ่านรหัสผ่านหลายระดับ และไม่มีกระบวนการส่งรหัสผ่านส่วนหลังผ่านโค้ด Client-side โดยตรง
* **ประสิทธิภาพสูงสุดในการทำงาน (Maximized Efficiency)**: ลดเวลาค้นหา URL หรือระบบต่าง ๆ ในแต่ละวันลงกว่า 90%
* **ทันสมัย ล้ำยุค (Highly Prestigious & High-Tech UI)**: ดีไซน์หรูหราแบบ Minimal-Space ปัดแต่งแสงเงา นีออน และสีสันที่สบายตาเป็นมิตรต่อสายตาเจ้าหน้าที่ผู้เฝ้าหน้าจอกลางคืน ยกระดับเทคโนโลยีของฝ่ายเภสัชกรรมและภาพลักษณ์องค์กรของโรงพยาบาลกรุงเทพพิษณุโลก
* **ความพร้อมใช้งาน 100% (High Availability)**: มั่นใจได้ว่าพอร์ทัลหลักจะไม่ล่มตามระบบอินเทอร์เน็ตภายนอก ด้วยสถาปัตยกรรมการถอยร่นฐานข้อมูลท้องถิ่นออฟไลน์ (Local fallback engine)
